:::::::::Para criar um cartão::::::::::
enviar req POST para
/createCreditCard

necessário headers.
Exemplo de Headers:

x-api-key
zadKLNx.DzvOVjQH01TumGl2urPjPQSxUbf67vs0

{
    "workerIdentifier": 1,
    "cardType": "transport"
}

/activateCreditCard

Exemplo de envio:

Body
{
	"id":1,
	"securityCode": "123",
	"password":"1234"
}
